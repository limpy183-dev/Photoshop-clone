================================================================================
  COMPREHENSIVE IMPLEMENTATION GAPS REPORT
  Browser Photoshop Clone — Next.js 16 / React 19 / TypeScript / Canvas
  Generated: May 25, 2026
================================================================================

This report catalogs every Photoshop feature that is either:
  A) NOT IMPLEMENTED AT ALL (zero code, placeholder, or only type stubs)
  B) PARTIALLY IMPLEMENTED (usable or approximation status, with remaining gaps)

EXCLUDED from this report — per BOUNDARIES.md — are:
  - Adobe cloud/AI services (Generative Fill, Neural Filters, Sensei, Firefly)
  - Adobe account/auth, Creative Cloud sync, Adobe Fonts/Stock APIs
  - Native plugin runtimes (UXP, CEP, 8BF binary execution)
  - Exact Photoshop engine parity (brush/text/filter/PSD internals)
  - Production color/prepress parity (certified CMM, trapping, press-ready output)
  - Professional codec output beyond MediaRecorder (ProRes, DNxHR)
  - Browser hardware limits (canvas max, heap limit, GPU texture size)

Every gap listed below IS implementable with pure browser-side code (HTML5
Canvas, Web Workers, WebGL, Web Audio, OPFS, etc.) — no external services or
native binaries needed.

  SECTION 3: PARTIALLY IMPLEMENTED — CAPABILITY REGISTRY (APPROXIMATION)
================================================================================

These features are registered in capabilities.ts with status "approximation"
and have documented limitations that are still browser-fixable:

  33. typography.3d-text — 3D Text Extrusion
      Gap: Glyph meshes are editable approximations, not exact font outline
      triangulations. Missing: proper font-outline-to-3D-mesh pipeline.
      Effort: Large

  34. 3d.advanced-import-export — Advanced 3D Import/Export
      Gap: Exact proprietary vendor chunks (3DS/KMZ/U3D) are approximated.
      Missing: fuller U3D block support, more 3DS material/chunk coverage.
      Effort: Large

  35. 3d.raytrace-cross-section-print — Ray Trace, Cross Section, Print
      Gap: No GPU path tracer, vendor-grade physical renderer, or printer-
      driver integration. CPU ray-trace is sampled, not production-grade.
      Effort: Very Large (GPU path tracer = major sub-project)

  36. format.psd — PSD Round-Trip
      Gap: ag-psd hardcodes RGB/8bpc; high-bit/CMYK/Lab preserved as metadata
      only. Some adjustments (shadows-highlights, hdr-toning, desaturate,
      match-color, replace-color, equalize) use marker-name encoding.
      Missing: native PSD descriptor emission for all adjustment types;
      proper non-RGB/high-bit layered PSD export.
      Effort: Very Large (limited by ag-psd capabilities)

  37. psd.color-modes — PSD Color Modes & Bit Depth
      Gap: ag-psd only encodes RGB/8bpc on disk. Non-RGB modes are metadata-
      only. Missing: native CMYK/Grayscale/Lab/Multichannel PSD export.
      Effort: Very Large (requires custom PSD writer for layered color modes)

  38. psd.effects-adjustments — PSD Layer Effects & Adjustments
      Gap: 6 adjustment types use marker-name fallback instead of native
      ag-psd descriptors. Missing: native descriptor emission for all 22
      adjustment types.
      Effort: Medium

  39. psd.vector-text — PSD Vector, Shapes, Text, Paths
      Gap: Custom shape and extended text markers. Missing: fuller native PSD
      text descriptor coverage (warp, advanced OpenType, paragraph styles).
      Effort: Medium

  40. psd.channels-masks — PSD Channels & Masks
      Gap: Saved alpha channels go through hidden marker groups; spot channels
      use naming conventions. Missing: native PSD channel/mask resource
      emission through ag-psd where available.
      Effort: Medium

  41. psd.resources-metadata — PSD Image Resources & Metadata
      Gap: IPTC/EXIF not surfaced by ag-psd; smart object bytes capped at
      30MB; lossy print-settings mapping. Missing: IPTC/EXIF passthrough,
      larger smart object payloads, richer print-settings preservation.
      Effort: Medium

  42. format.psb — PSB Large Document
      Gap: Compositor still uses single HTML5 2D context. Missing: proper
      tile-only mode for PSBs exceeding canvas limits; consistent "open at
      reduced scale" UX for every import path.
      Effort: Medium-Large

  43. format.raw-dng — RAW/DNG
      Implemented: DNG-style TIFF/EP export, Camera Raw sidecar round-trip,
      source fingerprint reconciliation, snapshots/settings preservation, and
      local lens profile matching. Remaining production caveat: proprietary
      camera/lens profile databases and raw sensor mosaic reconstruction.

  44. format.openexr — OpenEXR
      Implemented: arbitrary-channel authoring, tiled/deep metadata records,
      and multipart manifests. Remaining production caveat: certified OCIO
      transforms and true HDR display are outside the browser canvas.

  45. format.pdf — PDF
      Implemented: multipage authoring/import, flattened previews, editable
      text/vector manifests, transparency group records, and annotations.
      Remaining production caveat: PDF/X and external-editor preservation of
      app manifests require dedicated layout/prepress tools.

  46. format.eps — EPS
      Implemented: broader safe-subset coverage for transforms, dash patterns,
      fill rules, text, paths, curves, and editable vector reconstruction.
      Remaining production caveat: arbitrary PostScript execution,
      separations, overprint, and font resolution stay out of scope.

  47. format.heif — HEIF/HEIC
      Implemented: HEVC-backed HEIC container authoring through a configured
      encoder callback, distinct from AVIF-backed HEIF export. Remaining
      production caveat: auxiliary/depth images and live-photo pairing.

  48. format.jpeg2000 — JPEG 2000
      Implemented: codestream/JP2/JPX/JPM authoring, alpha channel
      definitions, color boxes, ICC/profile metadata, and layer labels.
      Remaining production caveat: archival conformance validation.

  49. format.dicom — DICOM
      Implemented: compressed transfer-syntax metadata, overlays, richer
      metadata workflow, and explicit non-clinical validation labels.
      Remaining production caveat: clinical diagnostic validation and private
      hospital workflows.

  50. smart-object.linked — Linked Smart Objects
      Gap: No native file watcher or external linked-document sync daemon.
      Missing: could implement periodic file poll via File System Access API
      handle persistence.
      Effort: Medium

  51. workflow.content-aware-scale — Content-Aware Scale
      Gap: Large changes fall back to resampling; no protected skin-tone/
      object models. Missing: better saliency detection for protected regions
      in seam carving.
      Effort: Medium

  52. external.native-plugins — Plugin Runtime Compatibility
      Gap: UXP/CEP adapters expose a documented browser subset, not full
      Adobe host process. Missing: broader Action Manager descriptor coverage;
      deeper UXP DOM compatibility.
      Effort: Large
