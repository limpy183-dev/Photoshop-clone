TGA Export
  Current state:
  - TGA import only, no export
  What's missing (implementable):
  - TGA encoder: write TGA header (18 bytes) + pixel data in BGR(A) order
  - TGA is one of the simplest image formats to encode — 18-byte header
    followed by raw or RLE-compressed pixel data
  - RLE compression pass on pixel rows
  - Offer as export option alongside PNG/JPEG/WebP
  - Pure typed-array encoding, no external deps
  Effort: Small — TGA encoder is ~50 lines of code

PNM Export
  Current state:
  - PNM import only, no export
  What's missing (implementable):
  - PPM export: write ASCII P6 header + raw RGB bytes
  - PGM export: grayscale variant
  - PBM export: 1-bit black and white variant
  - All are trivial formats: header string + raw bytes
  - Offer as export options
  Effort: Trivial — PNM encoders are ~30 lines each
